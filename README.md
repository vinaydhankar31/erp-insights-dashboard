# Employee Performance & HR Analytics Dashboard

## Project Overview
This project presents an interactive data analytics dashboard built using Power BI to analyze employee performance, satisfaction levels, and workforce insights. The dashboard enables HR teams and management to make data-driven decisions by visualizing key employee metrics and identifying patterns in performance and satisfaction.

## Objectives
• Analyze employee performance ratings across different departments.  
• Understand employee satisfaction levels and engagement.  
• Identify correlations between education level, performance ratings, and employee satisfaction.  
• Provide HR managers with clear visual insights for workforce planning.

## Dataset
The dataset consists of multiple related tables used for HR analysis:

- Employee.csv – Employee demographic and performance data  
- EducationLevel.csv – Education categories of employees  
- PerformanceRating.csv – Performance rating metrics  
- RatingLevel.csv – Rating classification levels  
- SatisfiedLevel.csv – Employee satisfaction categories  

These datasets were integrated and modeled within Power BI to create relationships and meaningful insights.

## Key Dashboard Features
• Interactive visualizations for employee performance analysis  
• Employee satisfaction distribution analysis  
• Performance rating trends across departments  
• Education level vs performance comparison  
• KPI indicators for workforce insights

## Business Insights
The dashboard helps organizations:
- Identify high-performing employees
- Understand employee satisfaction trends
- Improve HR decision-making
- Support workforce planning and performance management
